# Сайт о фильмах
![alt text](assets/images/preview.png)
## архитектура сайта
**сайт вкллючает в себя следующие страницы:**
-главная - показывает все доступные фильмы и использует пагинацию (на каждой мтранице по 12 фильмов)
-ТОП-25 - показывает топ-25 всех фильмов
-актеры - показывает список всех актеров, по нажатию на актера всплывает окно с информацией о нем
-фильм-отображает выбранный фильм

## запросы для получения данных

**ccылка на api - http://127.0.0.1:3000 **

-`GET/films`- получение всех фильмов.
-`GET/actors`- получение фильма по его id
-`GET/films/{id}`- получение списка актёров
-`GET/actors/{id}`-получение актера по его id
-`GET/top25`- получение списка 25 лучших фильмов
